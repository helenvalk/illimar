3x − 7 > 2
💡 Vihje
Tõsta −7 paremale poole. Märk ei pöördu, sest jagad positiivse arvuga.
Samm 1 — Isoleeri x
3x − 7 > 2
3x > 2 + 7
3x > 9
Liidan mõlemale poolele 7.
Samm 2 — Jaga 3-ga
x > 3
Jagan positiivse arvuga → märk ei pöördu.
Samm 3 — Vastus intervalliga
x ∈ (3; +∞)
Avatud sulg — 3 ise lahendusse EI kuulu (märk on >, mitte ≥).

LihtneLineaarne võratus⚠️ Märgi pöördumine
Tähelepanu — jagamine negatiivse arvuga!
📋 Ülesanne
−2x + 5 ≤ 11
💡 Vihje
Lõpuks jagad −2-ga. Negatiivse arvuga jagades pöördub ≤ märk → ≥.
Samm 1 — Tõsta konstant
−2x ≤ 11 − 5
−2x ≤ 6
Lahutan mõlemalt poolelt 5.
⚠️ Samm 2 — Jaga NEGATIIVSE arvuga → märk pöördub!
x ≥ 6 ÷ (−2)
x ≥ −3
Jagasin −2-ga → ≤ muutub ≥-ks. See on kõige levinum viga!
Samm 3 — Vastus
x ≥ −3   ehk   x ∈ [−3; +∞)
Suletud sulg — −3 kuulub lahendusse (märk on ≥).
✅ Vastus
x ≥ −3   ehk   x ∈ [−3; +∞)

KeskmineRuutvõratus
Ruutvõratus — intervalltabel
📋 Ülesanne
x² − 5x + 6 < 0
💡 Vihje
Leia kõigepealt nullid (kus avaldis = 0). Siis vaata: parabolil a > 0, mis tähendab, et see on allapoole lahti. Kus on parabolil allpool x-telge?
Samm 1 — Leia nullid
x² − 5x + 6 = 0
(x − 2)(x − 3) = 0
x₁ = 2,   x₂ = 3
Faktoriseerin või kasutan diskriminanti.
Samm 2 — Paraboli suund
a = 1 > 0 → parabolil on miinimum
Parabolil on kuju ∪ (naerunägu)
Kuna a > 0, on parabolil allpool x-telge ainult nullide vahel.
Samm 3 — Intervalltabel
x < 2: (+) positiivne
2 < x < 3: (−) negatiivne ← see on lahend
x > 3: (+) positiivne
Võratus < 0 tähendab, et otsime negatiivseid väärtusi.
Samm 4 — Vastus
x ∈ (2; 3)
Avatud sulud — nullid ise EI kuulu lahendusse (märk on <, mitte ≤).
✅ Vastus
x ∈ (2; 3)

KeerukamAbsoluutväärtus
Absoluutväärtuse võratus — kaks juhtu
📋 Ülesanne
|2x − 3| ≤ 5
💡 Vihje
Reegel: |A| ≤ a tähendab −a ≤ A ≤ a. Kirjutan ühe kaksikõrratusena.
Samm 1 — Reegel |A| ≤ a
|2x − 3| ≤ 5
−5 ≤ 2x − 3 ≤ 5
|A| ≤ a muutub kaksikõrratuseks −a ≤ A ≤ a.
Samm 2 — Liida 3 igale osale
−5 + 3 ≤ 2x ≤ 5 + 3
−2 ≤ 2x ≤ 8
Samm 3 — Jaga 2-ga
−1 ≤ x ≤ 4
Jagasin positiivse 2-ga → märk ei pöördu.
Samm 4 — Vastus
x ∈ [−1; 4]
Suletud sulud — mõlemad otsad kuuluvad lahendusse (≤).
✅ Vastus
x ∈ [−1; 4]
