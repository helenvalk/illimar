💡 Kuidas neid materjale kasutada
🎯
Alusta näidislahendusest
Iga materjalis on täielik näidislahendus koos sõnalise osaga. Loe see läbi enne oma ülesande lahendamist.
💬
Kasuta fraasipanka
Iga teema lõpus on valmis laused. Eksamil ei pea sõnastust välja mõtlema — vali sobiv fraas ja kohandad.
🔁
Struktuur: antud → otsin → valem → asendus → vastus
See 5-sammuline struktuur töötab igas tekstülesandes. Harjuta seda kuni see muutub automaatseks.
✅
Kontrolli alati
Asenda saadud vastus algülesandesse tagasi. Kirjuta: „Kontrollin: kui x = __, siis..."
📋
Kokkuvõttetabel = kiirviide
Iga materjali lõpus on kokkuvõttetabel. Kui oled teema selgeks saanud, piisab ainult tabelist.
🧩
Muster enne arvutust
Enne kalkulaatorit: mis tüüpi ülesanne see on? Millise valemi see meenutab? Tuvasta muster, siis arvuta.
✍️ Üldine fraasipank — eksami jaoks

„Koostan võrrandi/süsteemi antud tingimuste põhjal."

„Kuna [põhjus], siis [järeldus]."

„Leiame tundmatu: ..."
„Kontrollin: kui x = __, siis..."
„Seega on [mida otsiti] = __ [ühik]."

„Kolmas nurk: C = 180° − A − B"
„Lõikan [kujundi] kaheks osaks..."
„Kogu [pindala/pikkus/summa]: S = S₁ + S₂"
„Negatiivse arvuga jagades pöördub märk."
„Nimetaja ei saa olla null, seega x ≠ ..."
„Leiame kriitilised punktid, kus avaldis = 0."
